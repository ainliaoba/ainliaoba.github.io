# [如果那天我再勇敢一点](https://ainliaoba.github.io/)

《如果那天我再勇敢一点》（遗憾错过）

我们之间没有吵架，没有背叛，甚至没有告别。

只是有一天，不再联系了。

你一直以为时间很多，可以慢慢来。喜欢不用说破，暧昧就很好；等稳定了再认真，等合适了再确定，等未来再计划。

可生活从不等人。

她换了城市，你忙于工作；她分享日常，你回得越来越晚；你想找个合适的时间说喜欢，却总觉得“改天更好”。

后来，她开始礼貌了。

消息变短，语气变客气，连笑容都像对朋友。你终于察觉不对，却又不知道从哪一句挽回。等你鼓起勇气，她已经在别人的故事里。

遗憾不是失去，而是本可以。

你会在某个普通的夜晚突然想起：如果那天我再主动一点，如果那次我没有沉默，如果我早点承认自己很在乎……

可人生没有“如果”。

有些人教会你心动，有些人教会你长大，而有些人，教会你勇敢要趁早。
